### üìå Project Description (EN)

This project delivers a full-cycle analysis of real-world e-commerce transactions from a UK-based online store. It goes beyond traditional EDA, aiming to replicate the full analytical pipeline ‚Äî from raw data to business-ready insights.

**Key objectives:**

* Clean and prepare the dataset (handle missing values, duplicates, returns, and anomalies)
* Calculate core product & business metrics: GMV, AOV, LTV, Items per Order
* Perform detailed cohort analysis (retention, revenue, cumulative LTV)
* Run hypothesis testing (log-transformed t-tests, z-tests, bootstrapping)
* Build a final interactive BI dashboard to communicate insights

> üí∑ All monetary values are presented in British pounds (¬£)

### üß† Summary & Insights (EN)

We focused our analysis on two comparable markets ‚Äî France and Germany ‚Äî and identified key differences:

* France has significantly more orders per user than Germany (t-test, p < 0.05)
* Average product price is \~5.6% lower in France (t-test, p < 0.01)
* High-value orders are more common in France (z-test, p < 0.01)
* Log transformations and bootstrap methods helped deal with skewed distributions
* Cohort retention is weak: most users make purchases only in their first month
